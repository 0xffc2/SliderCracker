# 滑块验证码 js 破解

* 极验2
* 安居客(单IP大量请求后滑块验证会变成绘制手势验证)
* 美团
* 京东(轨迹算法需要重写)
* 易盾(轨迹算法通过率较低, 可自行改写)
* 云片(验证码图像处理缺口定位成功率有待提高)
* 数美(说明: 当接口返回数据中 riskLevel 字段为 PASS 即为验证通过, 在之后的请求中携带 rid 即可)
* 携程(只实现了滑块验证部分, 通过率极高, 控制频率不会出现后续点选验证)
* 搜狐
